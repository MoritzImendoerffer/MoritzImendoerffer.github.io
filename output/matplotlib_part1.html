Pretty plots with Matplotlib part 1

Date Do 22 März 2018 Tags python / matplotlib

In designed experiments it is quite common to have multiple measurements at the same conditions (duplicates, triplicates etc.). From the perspective of data analysis that means, multiple points might share the same location on the x axis but the y values are different. There are many options which could be used for plotting, some of which are: Matplotlib style plots, pointplot in seaborns or time series plot in seaborn

For some datasets I developed my own way of plotting things. Especially, while looping over a grouped dataframe. Let me share with you my solution:

ng = 5 # number of groups
nr = 4 # number of repetitions

# time
x = np.arange(0, 10)

# represent experimental conditions e.g pH, Temperature etc.
a = np.linspace(0.1, 1, ng)
b = np.linspace(0.5, 1.5, ng)

# repeat each group nr times
a_rep = np.tile(a, nr)
b_rep = np.tile(b, nr)
# get x in the proper shape
x_rep = np.tile(x, (1,nr)).T

# for every condition, a sample has been withdrawn len(x) times
p = list(it.product(zip(a_rep, b_rep), x_rep))

# make a dataframe out of it
df = pd.DataFrame([np.concatenate(item) for item in p])
df.columns = ['a', 'b', 'time']

# make data and add noise
df['y'] = df.a*df.b*df.time/(1+df.a*df.time) + 0.1* np.random.normal(1,0.6, len(df))

My solution for this problem was to create functions which group the datapoints by the common key (x or time in this case):

In [227]:
def my_grouper_mean(y, key):
    """ Accepts key value vectors takes the median all values sharing the
        same key (e.g. time etc.)
    :param y vector containint the values to sum
    :param key some sort of key (same length as y) which defines the groups

    e.g y = [1, 3 ,5, 6, 7, 8]
        key = [1, 1, 1, 2, 2, 2]
    """
    y_ = np.array(y)
    key_ = np.array(key)
    # extract all unique groups
    unique_groups = np.unique(key_)
    # list for storing the sums
    values = []
    # iterate over all unique group values and append the sum of the groups
    for group in unique_groups:
        values.append(np.mean(y_[key_ == group]))
    return (unique_groups,np.array(values))

def my_grouper_std(y, key):
    """ Accepts key value vectors takes the median all values sharing the
        same key (e.g. time etc.)
    :param y vector containint the values to sum
    :param key some sort of key (same length as y) which defines the groups

    e.g y = [1, 3 ,5, 6, 7, 8]
        key = [1, 1, 1, 2, 2, 2]
    """
    y_ = np.array(y)
    key_ = np.array(key)
    # extract all unique groups
    unique_groups = np.unique(key_)
    # list for storing the sums
    values = []
    # iterate over all unique group values and append the sum of the groups
    for group in unique_groups:
        values.append(np.std(y_[key_ == group]))
    return (unique_groups,np.array(values))

Together with fill_between, a pretty nice plot can be created:

In [247]:
gr = df.groupby(['a', 'b'])

fig, ax = plt.subplots(ncols=1, figsize=(8,6))
for index, item in enumerate(gr):
    name, val = item
    
    # the function returns unique x values and the mean at each x value
    xm, ym = my_grouper_mean(val.y, val.time)
    
    # the function returns unique x values and the standard deviation at each x value
    xstd, ystd = my_grouper_std(val.y, val.time)
    
    ax.plot(xm, ym, '-o')
    ax.fill_between(xm, y1=ym-ystd, y2=ym+ystd, alpha=0.5)
    
ax.set_title('Much nicer looking plot')
ax.set_xlabel('Sampling points e.g. in time')
ax.set_ylabel('Artificial data')
